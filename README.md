# matheuslmarchetti-financial-data-pipeline-as-a-service
Pipeline de dados financeiros automatizado com coleta diária (yFinance, FRED e RSS), validação, versionamento, armazenamento em PostgreSQL e API REST versionada com autenticação JWT, logs estruturados e observabilidade. Projeto focado em práticas reais de Engenharia de Dados e APIs profissionais.
